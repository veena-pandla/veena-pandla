## 🔨 Featured Projects

### [ArchiVision](https://github.com/veena-pandla/archivision) — 2D Floor Plan → 3D Model

Architecture firms were waiting weeks for CAD teams to manually convert 2D blueprints. I built a CV pipeline to automate it.

- YOLOv8 for wall/door/window/furniture detection (trained on 500+ annotated floor plans)
- OpenCV preprocessing for real-world scan messiness (skew, shadows, faded lines)
- Blender Python API for 3D generation — got processing time from 8 hours → under 30 minutes
- Flask web interface so architects could use it without touching code

`Python` `YOLOv8` `OpenCV` `TensorFlow` `Blender API` `Flask`

---

### [Cloud-Native Microservices Lab](https://github.com/veena-pandla/microservices-lab) — Distributed Systems Platform

Built to understand how production distributed systems actually work at scale. Three-service platform: API Gateway, User Service, Analytics Service.

- gRPC inter-service communication (60% payload reduction vs REST in benchmarks)
- Cloud Spanner with interleaved schemas — fixed N+1 problem, query latency 100ms+ → <10ms
- Bigtable row key design optimized for time-range queries
- Full CI/CD with GitHub Actions, Trivy security scanning, canary deploys on GKE

`C#` `.NET 6` `gRPC` `Docker` `Kubernetes` `GCP` `Bigtable` `Cloud Spanner`
